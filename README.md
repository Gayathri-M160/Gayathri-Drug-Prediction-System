# Drug Prediction System

Author: Gayathri M

This is a Machine Learning based Drug Prediction System developed by
Gayathri M. The system predicts suitable drugs based on patient
medical parameters using Python and Flask.
FrontEnd - HTMl,CSS,jinja2 template

Problem Statement: 
* Patients often face delays in receiving proper medication after laboratory tests due to limited doctor availability, frequent hospital visits, and lack of immediate guidance. 
* Existing systems mainly focus on drug–target research and do not provide patient-specific drug prediction, doctor verification, or prescription management. 
* Therefore, there is a need for a system that can predict suitable drugs from patient lab parameters and enable doctors to review and update prescriptions remotely, ensuring faster and safer treatment support.

Features:
* Predicts drugs based on real patient lab parameters(Age, BP, Cholesterol, Na-K values, Sugar, BMI, etc.).
* Uses Decision Tree model trained on medical dataset.
* Provides web portal for patients to enter lab values & get drug prediction.
* Adds Doctor Portal →✅ 
* View patient history✅ 
* Modify/update predicted drug✅
* Add notes & prescriptions✅ 
Share via WhatsApp / Email / Hard copy
* Supports patients under regular monitoring who cannot frequently visit the hospital.
Final medication only after doctor verification → ensures safety
